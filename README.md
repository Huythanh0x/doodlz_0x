## doodlz_0x - Simple drawer

This project follow a lesson from [this book](https://www.amazon.com/Android-Programmers-App-Driven-Approach-Developer/dp/0134289366)

**Some feature of this app:**

- draw with mulit-touch
- change color
- change witdh of pen
- erase drawing
- save to image
- print(unstable)
